# Video Motion Monitor

Приложение на Qt и OpenCV для обработки видео в реальном времени, визуализации FPS и уровня движения.

</div>

## Основные возможности

- Переключение между источниками: веб-камера или локальный видеофайл
- Фоновое вычитание (KNN) с подсветкой движущихся объектов
- Графики `FPS` и `Motion Level` на базе Qt Charts
- Поток обработки вынесен в отдельный `QThread` для плавного UI
- Готовые тестовые ресурсы в `Resources/` для быстрого старта

## Быстрый старт

### Зависимости

- Qt 5.15+ (модули Widgets, Concurrent, Charts)
- OpenCV 4.x
- CMake или qmake (в проекте используется qmake-файл `videoEditor.pro`)

### Сборка через qmake

```bash
qmake videoEditor.pro
make -j$(nproc)
./videoEditor
```

### Альтернативный способ: Qt Creator

1. Откройте `videoEditor.pro`
2. Выберите комплект Qt (Desktop)
3. Нажмите `Configure Project`, затем `Build` и `Run`

## Обзор интерфейса

| Блок | Описание |
| --- | --- |
| Панель видео | Отображение исходного и обработанного кадра (с подсветкой движения) |
| Выбор режима | `Камера` или `Из файла`, отдельные панели управления |
| Контроль воспроизведения | Кнопки `Старт`, `Пауза`, `Стоп`, выбор файла |
| Аналитика | Графики `FPS Monitor` и `Motion Monitor` с автообновлением |

## Архитектура

- `MainWindow` — координирует UI, таймеры и обмен сигналами
- `FactoryUI` + `AbstractUI` — единый интерфейс для источников (`FromCamUI`, `FromFileUI`)
- `VideoProcessor` — обработка кадров в отдельном потоке, расчёт уровня движения
- `FramePlotWidget`, `MotionPlotWidget`, `FPSCollector` — вспомогательные виджеты для метрик

## Набор ресурсов

- `Resources/test_video.mp4` — демо-ролик
- Дополнительные изображения для экспериментов с распознаванием (`cards.jpg`, `shapes.png` и др.)
- При необходимости подключите свои файлы через кнопку `Выбрать файл`
